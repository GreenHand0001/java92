# 图书管理系统简介 java92

## 项目概述

图书管理系统是基于JavaEE技术栈的一个ssm+vue的前后端分离项目。项目采用MVC三层体系结构，结合了Spring、SpringMvc、Mybatis与Vue.js技术，通过Maven进行项目管理和构建。

## 技术栈

- 前端：Vue.js
- 后端：Spring、SpringMvc、Mybatis
- 数据库：MySQL
- 项目管理：Maven

## 功能模块

- 用户管理
- 图书管理
- 借阅管理
- 统计分析
- 系统设置

## 系统角色

- 系统管理员
- 图书管理员
- 普通用户

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL 5.7+
- JDK版本：1.8+
- 需安装Node.js（前端开发环境）

## 部署步骤

1. 配置数据库环境
2. 导入项目SQL文件
3. 使用Maven打包后端代码
4. 部署前端静态资源到Web服务器
5. 启动后端服务

- 前后端分离设计，便于分工与维护
- 基于B/S结构的Web系统，易于拓展与使用
